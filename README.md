# 字幕清理工具 Subtitle Cleaner

> 去除字幕時間軸、合併斷句，一鍵產出乾淨的 script，方便丟給 AI 翻譯。

🔗 **[立即使用](https://hui0315.github.io/subtitle-cleaner/)**

---

## 支援格式

| 格式 | 範例 |
|------|------|
| 時間戳文字 | `0:00` / `0:01:30` + 下一行文字 |
| SRT | 標準 `.srt` 字幕檔 |
| WebVTT | 標準 `.vtt` 字幕檔 |
| 純文字 | 每行一句的純文字 |

## 使用方式

1. 貼入或上傳字幕檔（`.srt` `.vtt` `.txt`）
2. 按「清理字幕」
3. 複製整理後的 script
4. 搭配提示詞「**原句翻譯成台灣中文，不整理。**」貼到 ChatGPT / Claude 翻譯

## 特色

- 純前端，無需安裝、無需登入、不上傳任何資料
- 自動偵測字幕格式
- 顯示詞數與 token 估算，方便控制 AI 輸入量

## 本地使用

直接下載 `index.html`，在瀏覽器開啟即可，無需伺服器。
